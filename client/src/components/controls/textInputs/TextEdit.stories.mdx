# TextEdit

A debounced &lt;textarea&gt; that can optionally limit the number of allowable
characters and display how many are left.

<Canvas withToolbar>
  <Story
    decorators={[withRegistrationMarks, withStore()]}
    name="sample"
  >
    <TextEdit
      maxLength={number('maxLength', 140)}
      maxRows={number('maxRows', 6)}
      onChange={action('onChange')}
      showCharsLeft={boolean('showCharsLeft', false)}
      style={{ maxWidth: 350 }}
    />
  </Story>
</Canvas>
